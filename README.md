# DER3 Frontend

This repository contains the React/Vite frontend for DER3.
The ASP.NET Core backend lives under [`backend/`](backend).

## Run Locally

1. Install frontend dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) if needed.
3. Start the frontend:
   `npm run dev`

For backend execution and deployment settings, use the project under [`backend/DER3.Api`](backend/DER3.Api).
